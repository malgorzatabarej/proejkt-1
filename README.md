# PROJEKT 1
## **Transformacje**
Dzięki temu programowi, można transformować współrzędne, w zależnosci od potrzeb użytkownika.

Dostępne są następujące elipsoidy:
```
GRS80
WGRS84
KRASOWSKIEGO
```

Dostępne są następujące transformacje:
```
XYZ do BLH
BLH do XYZ
XYZ do NEU
BL do PL2000
BL do PL1992
```
